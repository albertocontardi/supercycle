## Retrospective

The `retrospective` skill and `using-supercycle` bootstrap skill are original contributions by Alberto Contardi, released under the MIT License.

The core insight behind Retrospective: dormant skills are not waste to be pruned — they are institutional memory. If a recurring error pattern exists that a dormant skill could resolve, the skill gets reactivated (description updated) rather than archived.
