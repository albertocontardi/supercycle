# SuperCycle — Gemini CLI Setup

SuperCycle is active. You have access to structured development skills and an end-of-session retrospective system.

## How to use skills

Use the `activate_skill` tool to load skills before acting. If you think there is even a 1% chance a skill applies, load it.

Available skills are in the `skills/` directory. Key skills:

- `brainstorming` — before any creative work or new feature
- `writing-plans` — before implementation, after design approval
- `systematic-debugging` — before proposing any fix
- `test-driven-development` — during implementation
- `verification-before-completion` — before claiming work is done

## End of session

Always run `/retrospective quick` before ending a session. It takes ~1 minute and captures patterns for the next session.

## Skill discipline

If a skill applies to your task, you must use it. This is not optional.
Read the full `using-supercycle` skill above for details.
